🚀 Student Leave Management System

A simple and user-friendly web application that allows students to apply for leave and teachers to manage those requests efficiently.

🔗 **Live Demo:** https://leave-management-system-rr13.onrender.com/
 ✨ Features

👩‍🎓 Student Panel

* Apply for leave easily
* Check leave status anytime
* View updates after teacher approval/rejection

👩‍🏫 Teacher Panel

* View all leave requests
* Approve or reject requests
* Instantly update student leave status

 🔄 How It Works

1. Open the application and select **Student Login**
2. Enter your details and apply for leave
3. Go back to the home page and select **Teacher Panel**
4. View submitted leave requests
5. Approve or reject the request
6. Return to **Student Panel**
7. Enter the same name to check updated leave status

⚙️ Tech Stack
* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Database: (Currently using in-memory storage for demo)
* Future Scope: MongoDB for persistent data storage

 Note
This project currently uses a **lightweight in-memory storage approach** for demonstration purposes.
Data will reset when the server restarts.

🔜 The system is designed to be extended with **MongoDB** for permanent data storage.
